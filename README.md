# Fillet

Lean yet delicious combination of Minitest and Capybara for Acceptance BDD.

Inspired by [Steak](https://github.com/cavalle/steak).

## Installation

Add this line to your application's Gemfile:

    gem 'fillet'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install fillet
